## Installation et Exécution de l'Application

### Backend (Flask)

1. **Installer les dépendances**:
   ```bash
   pip install -r requirements.txt
   ```
2. **Exécuter le serveur Flask:**
  ```bash
  flask run
  ```
  Par défaut, l'application sera accessible sur http://127.0.0.1:5000/.
